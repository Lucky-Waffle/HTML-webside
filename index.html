Applications We Have

The bug tracker system is built up by three modules

Bug Report

Users will be able to access two of the three modules. They will be able to submit reports in the bug reporting module while also being able to see the status of bugs in the Status module.

DevOps Report Modul

Status

The Status module is used to search for and view status of all features and bugs.

DevOps Status Modul

Developer

Developers and Administrators have their own module where they have full access over the reports submitted. All reports can individually be claimed or assigned to a developer to be investigated.
